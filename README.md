# Price-Tracker-Python

A simple Python script to track product prices from e-commerce websites.

## Features
- Scrapes product names and prices using BeautifulSoup and Requests.
- Saves the extracted data into a CSV file for analysis.

## How to use
1. Install requirements: `pip install requests beautifulsoup4 pandas`
2. Run the script: `python price_tracker.py`

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
